The UAC project.

The UAC project is a project for windows to make an Universal Administrative Command prompt simmilar to cmd.exe and is written in batch(windows batch file).
It now only features an input screen but does have a function to listen if UAC is run as Administrator. there are some goals i want to achieve for this time being:

goals ▪️Plugins support
▪️actual usefull commands ex. disk configuring and registry settings.

status:unfinished ❌

bugs:

The bugs that i come across are all visual, they dont inpact how UAC works. But they can impact how people will intereact with UAC.
One bug im comming across is that the font and font size change when UAC is run in cmd.exe or conhost.exe thats why i reconmend using windows terminal
(wt.exe) for using UAC. and if you run it as Administrator via the buildt in admin command it will also restart UAC in Windows Terminal as Administrator.
